# CampusRide
CampusRide est une application de covoiturage destinée à la communauté du Collège La Cité (Ottawa). Elle permet aux étudiants, enseignants et employés de proposer, rechercher et réserver des trajets domicile–campus de manière sécurisée, via une application web et une application mobile Android (PWA).
